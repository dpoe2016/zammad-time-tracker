## [3.3.0] - 2025-10-31

Added comprehensive sprint planning functionality to organize and track agile sprints:

**Sprint Management**
- Create sprints with name, start/end dates, and goals
- Start and complete sprints with automatic state transitions
- View sprint history and track completed work
- Only one active sprint at a time

**Backlog & Planning**
- Visual backlog view with all unassigned tickets
- Drag-and-drop interface to assign tickets to sprints
- Real-time filtering by state and search
- Automatic ticket organization

**Time Estimation**
- Set estimated hours for each ticket in sprint
- Track total capacity and estimated work
- Edit estimates at any time
- Visual progress indicators

**Sprint Statistics Dashboard**
- Total tickets in sprint
- Completed ticket count
- Total estimated hours
- Progress percentage

**Technical Implementation**
- New IndexedDB stores for sprints and assignments (schema v2)
- Sprint manager module for data operations
- Dedicated sprint planning UI page
- Integration with existing dashboard and time tracking
- Responsive design with mobile support

**Documentation**
- Comprehensive sprint planning guide (SPRINT_PLANNING.md)
- Quick start tutorial (QUICKSTART_SPRINT.md)
- Updated README with sprint features

**Files Added**

- src/sprint-planning.css - Sprint-specific styles
